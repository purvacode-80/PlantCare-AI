## PlantCare AI

PlantCare AI is a web-based plant disease detection system 
built using Transfer Learning (MobileNetV2) and Flask.

## Features
- Upload plant leaf image
- AI-based disease classification
- Treatment recommendations
- Clean web interface

## Model
- MobileNetV2 (Transfer Learning)
- Fine-tuned on New Plant Diseases Dataset

## Tech Stack
- Python
- TensorFlow / Keras
- Flask
- HTML / CSS

## Team Project
Built for AIML Domain Submission.
